== Description ==

The Fielder is an collection of various custom fields that you can seamlessly use with The Assely framework and it's singularities. A considerable number of fields types allows for shaping your application metadata as much as you like.

### Interested? Take a look at documentation.
* [Project website](https://assely.org)
* [Documentation](https://assely.org/docs/fielder-installation)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/assely-fielder` directory, or install the plugin through the WordPress plugins screen directly
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Follow up [documentation](https://assely.org/docs/fielder-installation)
